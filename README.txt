================================================================================
ABOUT
================================================================================

Cocktail is an actionscript full stack framework built to deploy fast rich
internet applications with a robust & high level interactive and data driven
development style.

================================================================================
VERSION
================================================================================

- 03/05/2010 - 0.3 Unstable
- 03/05/2009 - 0.1 Alpha

We have maintained an AS2 version for years in a svn repo. 

That version was a sketch/lab of all the ideas of grandeur presented in the 
current version of the project.

In the end we decided to put that aside and start from scratch, with a nicer 
and cleaner new foundation.
